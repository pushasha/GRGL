Grgl.Item Namespace
 
Classes
  Class Description
Inventory
Represents the personal inventory of a character, or the contents of a container
Interfaces
  Interface Description
IConsumable
Items that are consumable by a player should implement this interface.
IEquippable
Items that are equippable by a character should implement this interface.
IInventory
Represents the personal inventory of a character, or the contents of a container
IInventoryItem
Represents the quantity of items in an inventory using an integer value, and a dictionary of the item instances
IItem
Represents an in-game that can be possessed by a character.
IUsable
Items on which a "use" action can be performed should implement this interface.
IWeapon
Items that act as equippable weapons should implement this interface.
IWearable
Equippable items that are considered wearable should implement this interface.
